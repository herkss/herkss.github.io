클래스 정의 및 객체 생성 방법.

인터페이스의 정의 및 사용법.
Mixins의 개념과 적용 예.
예제 코드:

```dart
코드 복사
class Animal {
  void speak() {
    print('Animal speaks');
  }
}

class Dog extends Animal {
  @override
  void speak() {
    print('Woof!');
  }
}

void main() {
  Dog dog = Dog();
  dog.speak();  // Output: Woof!
}
```






2회차: Dart 기본 문법과 데이터 타입
1. 변수 선언 및 데이터 타입
변수 선언

Dart에서 변수를 선언할 때는 var, final, const 키워드를 사용할 수 있습니다.
var

타입 추론을 사용하여 변수의 타입을 자동으로 결정합니다.
```dart

var name = 'Alice'; // String 타입으로 추론
var age = 30;      // int 타입으로 추론
```
final

한 번만 값을 할당할 수 있으며, 런타임에 초기화됩니다.
```dart

final city = 'Seoul';
const
```
컴파일 타임에 값을 결정하며, 불변입니다.
```dart

const pi = 3.14;
```
데이터 타입

Dart의 기본 데이터 타입:
int: 정수
double: 부동 소수점 숫자
String: 문자열
bool: 불리언 (true/false)
예제 코드:

```dart

void main() {
  int age = 25;
  double height = 175.5;
  String name = 'John';
  bool isStudent = true;

  print('Name: $name, Age: $age, Height: $height, Is Student: $isStudent');
}
```
2. 제어문
if 문

조건에 따라 실행할 코드를 선택합니다.
```dart

void main() {
  int score = 85;

  if (score >= 90) {
    print('A');
  } else if (score >= 80) {
    print('B');
  } else {
    print('C');
  }
}
```
switch 문

여러 조건을 쉽게 관리할 수 있는 방법입니다.
```dart

void main() {
  String grade = 'B';

  switch (grade) {
    case 'A':
      print('Excellent');
      break;
    case 'B':
      print('Good');
      break;
    case 'C':
      print('Average');
      break;
    default:
      print('Invalid grade');
  }
}
```
3. 반복문
for 문

정해진 횟수만큼 반복합니다.
```dart

void main() {
  for (int i = 0; i < 5; i++) {
    print('Count: $i');
  }
}
```
while 문

조건이 참인 동안 반복합니다.
```dart

void main() {
  int count = 0;

  while (count < 5) {
    print('Count: $count');
    count++;
  }
}
```
do-while 문

최소 한 번은 실행된 후 조건을 확인합니다.
```dart

void main() {
  int count = 0;

  do {
    print('Count: $count');
    count++;
  } while (count < 5);
}
```
4. 함수 정의
함수의 기본 구조

함수는 입력값(매개변수)과 결과값(반환값)을 가질 수 있습니다.

```dart

int add(int a, int b) {
  return a + b;
}

void main() {
  int result = add(5, 3);
  print('Result: $result');
}
```
선택적 매개변수

선택적 매개변수를 사용하여 기본값을 설정할 수 있습니다.
```dart

void greet(String name, {String greeting = 'Hello'}) {
  print('$greeting, $name!');
}

void main() {
  greet('Alice'); // Hello, Alice!
  greet('Bob', greeting: 'Hi'); // Hi, Bob!
}
```
가변 매개변수

여러 개의 매개변수를 받을 수 있는 방법입니다.
```dart

void printNumbers(int a, int b, [int? c]) {
  print('Numbers: $a, $b, $c');
}

void main() {
  printNumbers(1, 2);        // Numbers: 1, 2, null
  printNumbers(1, 2, 3);     // Numbers: 1, 2, 3
}
